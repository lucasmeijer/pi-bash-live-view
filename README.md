# bash-terminal pi extension

Minimal pi extension scaffold for iterating on a widget that appears above the prompt.

## Files

- `index.ts` — extension entrypoint

## Install for local iteration

This repo is symlinked into pi's global extension directory:

- `~/.pi/agent/extensions/bash-terminal -> /Users/lucasmeijer/src/tries/2026-03-17-bash-terminal`

After editing, reload pi with:

```text
/reload
```
